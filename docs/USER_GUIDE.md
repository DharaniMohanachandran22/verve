# User Guide: Getting Started with Verve

Welcome to **Verve**, your modern collaborative workspace for project management. This guide will walk you through the core concepts, roles, and features of the application.

---

## 🌟 Purpose of Verve
Verve is designed to streamline team collaboration by providing a highly visual and interactive Kanban board. It bridges the gap between complex project management tools and simple to-do lists, offering a "glassmorphic" interface with real-time-like feedback.

## 👥 Roles & Permissions

Verve uses a simple but effective role-based system to ensure data integrity and security.

### 1. Board Owner
- **Creation**: The user who creates a board is the default owner.
- **Full Control**: Can create, update, or delete the board, lists, and cards.
- **Invitation**: Can invite other users to the board via email/invitation links.
- **Settings**: Can modify board visibility and general metadata.

### 2. Board Member
- **Collaboration**: Can view the board and add/edit cards and comments.
- **Movement**: Can move cards between lists to reflect progress.
- **Interaction**: Can participate in discussions via the card comments system.

### 3. Board Viewer
- **Read-Only Access**: Can view boards, lists, and cards to track progress.
- **Transparency**: Can see comments and activity but cannot add or edit any content.
- **Stakeholder Focus**: Designed for managers or stakeholders who need visibility without direct interaction.

---

## 🛠️ Core Features

### 📅 Kanban Boards
The heart of Verve. Organize your work visually using columns (Lists) and items (Cards). Each board represents a specific project or workflow.

### 🗂️ Nested Lists & Cards
- **Lists**: Represent stages in your workflow (e.g., "To Do", "In Progress", "Done").
- **Cards**: Represent individual tasks. Cards can contain titles, descriptions, and comments.

### 🔔 Smart Notifications
Stay updated with real-time-like alerts. You'll be notified when:
- You are added to a new board.
- Someone comments on a card you're watching.
- Important board changes occur.

### 🔍 Global Search
Find exactly what you need. The search feature allows you to quickly locate boards and specific cards across your entire workspace.

---

## 🚀 How to Use Verve: Step-by-Step

1.  **Register & Login**: Create your account and set up your profile.
2.  **Create a Board**: Start your first project by clicking "Create New Board" on the dashboard.
3.  **Invite Your Team**: Use the "Invite" button inside the board to add collaborators.
4.  **Define Your Workflow**: Create Lists for each stage of your process.
5.  **Add Tasks**: Create Cards for individual items.
6.  **Progress Tracking**: Drag and drop cards from one list to another as work moves forward.
7.  **Collaborate**: Add comments and track updates via the notification center.
